# DASH Over 50 METHOD™

AI-powered health tracking application for people over 50.

## Quick Start

### Prerequisites
- Node.js 18+ 
- npm

### Installation

\`\`\`bash
npm install
\`\`\`

### Development

\`\`\`bash
npm run dev
\`\`\`

Visita `http://localhost:5173`

### Build

\`\`\`bash
npm run build
\`\`\`

Genera la cartella `dist/` pronta per il deploy.

### Preview Build

\`\`\`bash
npm run preview
\`\`\`
